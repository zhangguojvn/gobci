## 介绍

Go 的 Oceanbase Oracle 模式驱动

## 安装

将官方提供的 libobclient 及 obci 包中的 libobci.so.* 和 libobclnt.so.* 放入 /usr/lib 目录

安装 GCC 后即可拉取驱动

```
go get github.com/zhangguojun/gobci
```
